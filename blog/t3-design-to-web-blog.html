  1. What a responsive site is, and why responsiveness is important?

    2. A responsive site is a site that changes in accordance to the size of the window or screen. Content of the website shrink when the window is made smaller and vice versa. In general responsive design focusses on maintaining the quality no matter how large or small the screen.

  2. What mobile first design is, and why it's important.

    3. Mobile first is focussing on designing a mobile compatible website, when creating a website for a computer. Web Developers in the past have typically focussed on creating a website for a computer screen, which makes it harder to design and create a mobile friendly website.
